# Backend

Express API for Kanya Vidyalay Manerajuri.

## Commands

```bash
npm install
npm run seed:admin
npm run dev
npm start
```

## Important Files

- `src/server.js`: starts the API
- `src/app.js`: Express app and routes
- `src/models`: MongoDB schemas
- `src/controllers`: API logic
- `src/routes`: REST routes
- `src/middleware`: authentication, roles, upload, validation, errors
